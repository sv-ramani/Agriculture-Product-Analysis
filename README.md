# Agriculture-Product-Analysis
# 🌾 Agriculture Production Analysis Dashboard | Power BI

## 📌 Overview
This project presents an interactive Power BI dashboard designed to analyze agriculture production data and deliver meaningful business insights. It helps users monitor key performance indicators, compare regional performance, and identify trends through interactive visualizations.

---

## 🎯 Objectives
- Analyze agriculture production performance across different regions.
- Monitor key business metrics using KPI cards.
- Compare revenue, profit, and ratings.
- Identify top-performing categories and states.
- Support data-driven decision-making through interactive reports.

---

## 📊 Features
- KPI Cards for Revenue, Profit, and Average Rating
- Region-wise and State-wise Performance Analysis
- Category-wise Revenue Distribution
- Profit Analysis by Region
- Product Rating Analysis
- Interactive Slicers for Region, State, and Category
- Decomposition Tree for detailed data exploration
- Treemap, Bar Charts, and Pie Charts for effective visualization

---

## 🛠️ Tools & Technologies
- Power BI Desktop
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling
- Data Visualization

---

## 📈 Key Insights
- Evaluated revenue and profit across multiple regions and states.
- Identified top-performing product categories.
- Compared product ratings to understand performance.
- Explored data interactively using filters and decomposition analysis.
- Created an intuitive dashboard to support business decision-making.
